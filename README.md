Node.js REST API для приложения TODO
=====================================

## Инициализация:


```bash
# Клонирование репозитория:
git clone git@github.com:renivi-eth/todo_api.git

cd todo_api

# Установка зависимостей:
npm ci

# Инициализация базы данных:
docker-compose up -d

# Запуск сервера:
npm run dev
```
